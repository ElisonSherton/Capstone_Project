Age Detection of Indian Actors

    This project is infact a practise hackathon on Analytics Vidhya which requires us to tell given a picture of an actor, is he young, middle-aged or old-aged. CNNs and transfer-learning were the methods employed in order to build this classifier.
    The base model used was a ResNet-50 model which is pre-trained on the ImageNet challenge data which is meant to classify a million images into a thousand different classes. After training for around 11 epochs, an accuracy of 88% was recorded on validation data and eventually the same accuracy was obtained on test data as well.
    The model was constructed and trained using fastai which is built on top of PyTorch and I stood 23rd in the overall hackathon as on 28th Feb 2020 and you can view the entire code and approach mentioned in detain at Github here.

Twitter User Analysis and Visualization.

    This project aims to understand the behavior of a user through their tweets and interaction with other users on a popular social media platform called twitter. I reached out to a friend of mine Shriram Dusane and he nodded his consent for the data to be used for the sake of this analysis.
    I extracted tweets only until 23rd Jan 2020, starting from when the user created his profile. Also, I have only considered tweets which the user sent out and not retweets.
    The analyses I have conducted include study of tweet acticity by hour of day, by week of year, most used hashtags in a year and so on. Data was extracted using python and visualized using Tableau. The visualization along with all it's interactive components can be viewed here.
